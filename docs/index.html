TibiaData Assets

Here we have our JSON assets that are almost static for the TibiaData API.

This repo (tibiadata-api-assets) contains tooling that generates the assets json files used by the tibiadata-api-go image.

Assets that are available in the files are worlds, towns, houses, creatures and spells.
